# 💇‍♂️ La Elegancia - Professional Hair Supply & Loyalty System

Este proyecto es una plataforma de E-commerce avanzada para estilistas y profesionales de la belleza. Incluye un sistema dinámico de gestión de productos, perfiles diferenciados (Cliente/Admin) y un programa de fidelización basado en puntos.

## ✨ Características Principales
*   **Gestión de Usuarios**: Registro de clientes y acceso especial para `usuario_admin`.
*   **Sistema de Lealtad**: Acumulación automática de 100 puntos por compras superiores a $50,000.
*   **Catálogo Categorizado**: Organización por Herramientas, Cuidado y Tratamientos.
*   **Canje de Premios**: Los usuarios pueden redimir 15,000 puntos por una peinilla profesional.
*   **Carrito de Compras Independiente**: Interfaz lateral (Sidebar) para no interrumpir la navegación.

## 🛠️ Tecnologías Utilizadas
*   **HTML5 / Tailwind CSS**: Estructura y diseño responsivo de alta gama.
*   **JavaScript (ES6+)**: Lógica de Programación Orientada a Objetos (POO).
*   **FontAwesome**: Iconografía profesional.

## 🚀 Instrucciones de Uso
1. Registre su usuario en la pantalla de inicio.
2. Si desea ver las opciones de gestión, ingrese con el nombre de usuario: `usuario_admin`.
3. Agregue productos al carrito; el sistema calculará automáticamente sus puntos al finalizar la compra.
